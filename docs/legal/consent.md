## 1. はじめに

「HPKI 電子署名ツール」（以下「本ツール」）は、医療・介護現場での PDF 文書への電子署名を支援する**公開ベータ版ソフトウェア**です。

本書は、本ツールを業務利用される事業所様に、本ツールの**特性・リスク・データの取扱い**をご理解いただき、納得の上でご利用いただくための同意書です。

---

## 2. 本ツールの特徴

- **HPKI カード**（医師・看護師カード）、または**マイナンバーカード**を使った電子署名
- 複数の PDF を一括して署名
- **PIN や PDF の内容は、ご利用パソコンの外（インターネット）に出ない設計**
- オープンソース（MIT License）で開発・公開

---

## 3. 公開ベータ版である旨

本ツールは現在**公開ベータ版**として提供されており、以下の可能性があることをご理解ください。

| 項目 | 内容 |
|------|------|
| **動作不良** | 想定外のエラー・処理失敗が発生する可能性があります |
| **互換性問題** | 一部のカードリーダー、OS、ドライバ、Acrobat バージョンとの組み合わせで動作しないことがあります |
| **データ損失** | 念のため、署名対象の PDF は事前にバックアップを取ってください |
| **法的有効性** | 電子署名の法的有効性は、ご利用のカード発行元（J-LIS / MEDIS 等）の運用ガイドラインに準じます。本ツールは技術的に署名を施す機能を提供しますが、その法的解釈は保証しません |
| **継続性** | ベータ版終了・正式版への移行に伴い、ファイルの互換性が変わる可能性があります |

## 4. データの取扱い

### 4.1 本ツールが**収集・送信しない**もの

以下の情報は、本ツール提供元（lifemate-inc）を含む**第三者には一切送信されません**：

- ☑ **PIN コード**（カード暗証番号）
- ☑ **PDF の内容**
- ☑ **署名証明書の中身**
- ☑ **患者・利用者の個人情報**
- ☑ **業務記録・カルテ等の医療情報**

これらはすべて、ご利用パソコンの内部（`127.0.0.1` ローカルアドレス）でのみ処理されます。インターネット経由で送信される瞬間はありません。

### 4.2 本ツールが**インターネット通信する**もの

以下は本ツールの動作上、外部へアクセスします（PIN や PDF とは無関係）：

- 📡 **GitHub Pages から HTML/CSS/JS の取得**（最初のページ読み込み時のみ）
- 📡 **アップデート確認**（GitHub Releases API への問い合わせ、週 1 回程度）
- 📡 **タイムスタンプ局（TSA）への問い合わせ**（署名時、PDF のハッシュ値のみ送信）
- 📡 **証明書失効リスト（CRL）の取得**（証明書検証用、公開情報のみ）

### 4.3 オプトイン項目

- ☐ **匿名利用統計**（将来実装予定。実装時には改めて同意を確認します）

### 4.4 ログファイルについて

本ツールはローカル PC 上に `bridge.log` というログファイルを作成します。このログには以下が含まれます：

- ✅ 操作時刻・エラーメッセージ
- ✅ 署名証明書の Subject DN・SHA-1 指紋（業務記録目的）
- ❌ **PIN は一切記録されません**
- ❌ **PDF の内容も記録されません**

ログは PC 内に残るのみで、外部に送信されることはありません。トラブル時に「診断情報をコピー」ボタンを押した場合、ログの末尾と OS 情報がクリップボードにコピーされ、ご利用者様の判断で配布元へメール添付等で送信できます。
