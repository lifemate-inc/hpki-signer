# 困ったときの簡単対応

「うまく動かない」と思ったときに、まず試すことを書いています。

3 つ試してダメだったら、**すぐに連絡してください**。

---

## まず深呼吸

慌てないで大丈夫。**データは消えません**。

PCを再起動するだけで直ることが多いです。

---

## 症状から対処を探す

### 🔴 アイコンをクリックしても何も起きない

**やってみる**：

1. もう一度ダブルクリック（早すぎたかも）
2. PC を再起動
3. デスクトップにアイコンが本当にあるか確認

それでもダメ：**連絡してください**

### 🔴 ブラウザが開いたけど「ブリッジに接続できません」

**やってみる**：

1. 10秒待ってから F5（ページ更新）
2. デスクトップアイコンをもう一度クリック
3. PC を再起動

それでもダメ：**連絡してください**

### 🔴 「カードリーダーが見つかりません」

**やってみる**：

1. USBに差し直す（カチッとつなぐ)
2. 別の USB の口に差す
3. PC を再起動
4. 画面の「📥 ドライバーをダウンロード」を試す

それでもダメ：**連絡してください**

### 🔴 「PIN が違います」

⚠️ **STOP！3回連続で間違えると、カードがロックされる可能性があります**

**確認すること**：

- 数字の `0`（ゼロ）と `O`（オー）を間違えていないか
- 大文字と小文字
- スペースが入っていないか

**PIN がわからなくなったら**：

- マイナンバーカード → **市役所窓口へ**
- HPKIカード → **発行元（医師会・看護協会など）へ**

⚠️ **適当に推測して入力しないでください**

### 🔴 ブラウザが開かない

**やってみる**：

1. デスクトップアイコンをもう一度クリック
2. 既にブラウザで開いていないか確認（タブ）
3. PC を再起動

### 🔴 「最新版あり」と出続ける

**やってみる**：

1. デスクトップアイコンをダブルクリック
2. ブラウザを完全に閉じる（×ボタンを全部押す）
3. もう一度デスクトップアイコンをダブルクリック

それでもダメ：**連絡してください**

### 🔴 「アップデートに失敗しました」と出た

これは**システムの保護機能**が働いて、自動で前のバージョンに戻った状態です。

**やること**：

1. **そのまま使い続けて OK**（古い版ですが動きます）
2. 開発担当に連絡（自動報告される予定）
3. 修正版が出るのを待つ

### 🔴 署名はできたけど、PDFを開くと「無効」と表示

これは **Adobe Acrobat Reader の設定問題**です。

**やってみる**：

1. PDF を一度閉じる
2. Acrobat Reader を完全に終了
3. もう一度 PDF を開く

それでもダメ：**連絡してください**（一緒に設定を直しましょう）

### 🔴 動作がとても遅い

**やってみる**：

1. 他のアプリを閉じる
2. PC を再起動
3. ファイル数が多すぎないか確認（一度に 100 個など）

## 「診断情報をコピー」を活用する

何が起きているかわからないとき、**画面の一番下**にある：

```
🔒 セキュリティの仕組み | 📋 診断情報をコピー | 💬 フィードバック
```

の **「📋 診断情報をコピー」** をクリック！

```
↓ クリック後 ↓

「✅ 診断情報をコピーしました。
   メールやLINEで送ってください。」と表示される。
```

そのまま：

- メールの本文に **貼り付け（Ctrl+V）**
- LINE のメッセージに**長押し→貼り付け**

これを担当者に送ってください。**原因がすぐにわかります**。
