# 毎日の使い方

初期設定が完了された方向けの、**日々のご利用ガイド**です。

---

## 1日の流れ

```
朝の準備
  ↓
カードリーダーに IC カードをセット
  ↓
通常通り業務（記録・書類作成）
  ↓
電子署名が必要なときだけ、下記の手順
```

---

## 電子署名のしかた（基本）

### 手順1：起動

デスクトップの 「**電子署名ツール**」 アイコンをダブルクリック。

ブラウザが自動で開きます。

```
┌──────────────────────────────────────┐
│  🩺 HPKI 電子署名ツール               │
│  ──────────────────────────────────  │
│  💳 使用するカードの種類              │
│  ┌──────────────┐ ┌──────────────┐   │
│  │   🏥         │ │   💳         │   │
│  │ HPKIカード   │ │ マイナンバー │   │
│  │              │ │   カード     │   │
│  └──────────────┘ └──────────────┘   │
└──────────────────────────────────────┘
```

### 手順2：カード種別を選ぶ

使うカードのほうを**クリック**して選択します。

### 手順3：PIN を入力

```
┌──────────────────────────────────────┐
│  🔑 カードのPIN番号                  │
│  ┌────────────────────────────────┐ │
│  │ ●●●●●●●●●                  👁 │ │
│  └────────────────────────────────┘ │
│  カードに設定した PIN を入力してください│
└──────────────────────────────────────┘
```

PIN がわからない場合は、カード発行時の書類でご確認ください。

### 手順4：フォルダを選ぶ

「**📁 フォルダを選択**」ボタンを押して、署名したい PDF が入っているフォルダを選びます。

```
選んだフォルダの中の PDF が一覧で表示されます

📂 訪問記録\
   ├ ☑ 記録_20260518_山田さん.pdf
   ├ ☑ 記録_20260518_佐藤さん.pdf
   ├ ☑ 記録_20260518_鈴木さん.pdf
   └ ☑ 計画書_2026_5月.pdf
   ─────────────────
   4 件の PDF が見つかりました
```

### 手順5：署名開始

「**✍️ 署名を開始する**」を押します。

```
┌──────────────────────────────────────┐
│  ⏳ 署名しています...                │
│  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ │
│  記録_20260518_山田さん.pdf          │
│  2 / 4件                              │
└──────────────────────────────────────┘
```

数秒〜数十秒で完了します。

### 手順6：完了

```
┌──────────────────────────────────────┐
│  🎉 完了しました！                    │
│  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ │
│  4 件の PDF に署名しました            │
│  ┌──────────────────────────────┐    │
│  │  🔄 もう一度署名する           │    │
│  └──────────────────────────────┘    │
└──────────────────────────────────────┘
```

署名済みファイルは、**同じフォルダ**の中に `_signed.pdf` が付いたファイル名で保存されます。

例：`記録_20260518_山田さん.pdf` → `記録_20260518_山田さん_signed.pdf`

---

## ファイル名のカスタマイズ

「**詳細設定**」を開くと、以下を変更できます：

| 設定 | デフォルト | 例 |
|------|-----------|---|
| ファイル名に追加する文字 | `_signed` | `_署名済み` |
| 元のファイルを上書きする | OFF | ON |

通常は変更不要です。

---

## 1日の終わり

### カードを抜く

⚠️ **業務終了時は必ずカードをリーダーから抜いてください**

理由：第三者が操作できてしまう可能性があるため。

### アプリは閉じてOK

ブラウザのタブは閉じて問題ありません。
次回起動時はまた最初から始めます。

---

## 効率的な使い方

### ① 同じフォルダにまとめてから一括署名

毎回ファイルを 1 つずつ署名するより、**1 日分まとめて**フォルダに入れて一括署名すると効率的です。

### ② カードは差したまま

業務中はカードを差したままで構いません。
ただし**席を離れるとき**は抜いてください。

### ③ ファイル管理のおすすめ

```
📁 訪問記録\
  ├ 📁 未署名\           ← 署名前の PDF をここに
  │  ├ 記録1.pdf
  │  └ 記録2.pdf
  └ 📁 署名済み\          ← 署名後に手動で移動
     ├ 記録1_signed.pdf
     └ 記録2_signed.pdf
```

---

## やってはいけないこと

| ❌ NG | 理由 |
|------|------|
| PIN を**何度も**間違える | カードがロックされます |
| カードを**そのままにして席を離れる** | 第三者に使われるリスク |
| **「フォルダ選択」で C ドライブのルート**を選ぶ | 全 PDF を署名しようとして処理が長時間に |
| 同じファイルを**何度も署名** | 不要な署名が増えるだけです |
| 業務時間外に**勝手に**インストーラを実行 | 一時的にご利用いただけなくなる可能性 |
