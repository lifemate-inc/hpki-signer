# HPKI電子署名ツール マニュアル全集

このフォルダには、本ツールに関わるすべての立場の方向けのマニュアルが入っています。
**まず自分の立場を確認**して、対応するフォルダのマニュアルから読み始めてください。

## あなたはどの立場ですか？

### 🎯 当社（lifemate-inc）営業担当の方

訪問看護ステーションへ本ツールを提案する役割の方は **[sales/](sales/)** を読んでください。

| 状況 | 読むもの |
|------|---------|
| 商品を初めて提案する | [`sales/01_product-pitch.md`](sales/01_product-pitch.md) |
| 提案中に質問された | [`sales/02_faq.md`](sales/02_faq.md) |
| 初めて訪問する | [`sales/03_first-visit.md`](sales/03_first-visit.md) |

### 🛠 当社（lifemate-inc）フォロー担当の方

ステーション設置後のサポートを担当する方は **[support/](support/)** を読んでください。

| 状況 | 読むもの |
|------|---------|
| 設置当日の流れを知りたい | [`support/01_onboarding.md`](support/01_onboarding.md) |
| お客様から「動かない」と連絡 | [`support/02_diagnostics.md`](support/02_diagnostics.md) |
| 自分で対応できない問題に遭遇 | [`support/03_escalation.md`](support/03_escalation.md) |
| 配布先を整理・追跡したい | [`support/04_user-management.md`](support/04_user-management.md) |

### 👨‍💻 ステーション側 技術責任者・IT担当の方

ステーションでツールの導入・管理を担当する方は **[tech-admin/](tech-admin/)** を読んでください。

| 状況 | 読むもの |
|------|---------|
| まずシステム全体像を理解したい | [`tech-admin/01_system.md`](tech-admin/01_system.md) |
| 詳しくインストールしたい | [`tech-admin/02_install-detail.md`](tech-admin/02_install-detail.md) |
| セキュリティ仕様を確認したい | [`tech-admin/03_security.md`](tech-admin/03_security.md) |
| 設定をカスタマイズしたい | [`tech-admin/04_customization.md`](tech-admin/04_customization.md) |

### 👩‍⚕️ ステーション側 現場担当（看護師・事務員）の方

実際に電子署名を行う現場の方は **[end-user/](end-user/)** を読んでください。

| 状況 | 読むもの |
|------|---------|
| 初めて使う | [`end-user/01_quick-start.md`](end-user/01_quick-start.md) |
